About

The battery on my laptop has degraded to the point that I can no longer just leave Chrome tabs open all willy nilly.

I decided to make this extension to use instead of using bookmarks, which feel like a much more serious commitment to me.

All the code for this extension is fully available on github. All your data stays in your browser and is never uploaded anywhere.

I intend to migrate this extension to Firefox and Safari eventually.

Made by rohan in 2020.
